# Linguo

Linguo is a platform that helps users learn words, terms, and definitions using a spaced repetition algorithm.

## About the project

Linguo is a backend-driven platform designed to help users learn words, terms, and definitions efficiently through structured repetition and personalized learning flows, with AI features being actively integrated to enhance personalization and learning experience.

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- S3 storage
- External API integrations
- JWT Authentication
- Unit & Integration Testing
- Onion Architecture
